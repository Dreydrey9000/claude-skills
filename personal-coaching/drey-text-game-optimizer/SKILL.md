# Text Game Optimizer

Transform draft messages into confident, action-oriented texts that project high value.

## When to Use

- User shares a draft text and asks for feedback
- User tends to over-explain in messages
- User wants to sound more confident
- User is in "interview mode" and needs to shift

## The Rules

### 1. BREVITY
Under 15 words unless logistics require more. If it takes you a paragraph to say "want to grab coffee?", there's a problem.

### 2. SPECIFICITY
Every message has a clear ask or next step. No messages that go nowhere.

### 3. NO HEDGE LANGUAGE
Remove: "maybe", "if you want", "no worries if not", "no pressure", "I was thinking perhaps"

### 4. DIRECT > CLEVER
Say what you mean. Don't wordplay or hint. Confident people are direct.

### 5. ONE MESSAGE
Never double-text while waiting for a response. Silence is data.

## Common Problems to Fix

| Problem | Example | Fixed |
|---------|---------|-------|
| Over-explaining | "Hey! So I was thinking, if you're not too busy this weekend and the weather is nice, maybe we could potentially grab coffee or something?" | "Coffee Saturday?" |
| Interview mode | "How was your week? What did you do? Any fun plans?" | "That restaurant Saturday. 7pm work?" |
| Approval seeking | "Hope that works for you! Let me know if not!" | [Delete it] |
| No clear ask | "It would be cool to hang out sometime" | "Thursday evening. Drinks at [place]?" |
| Excessive enthusiasm | "Omg that sounds AMAZING!!! I would LOVE that!!!" | "I'm in." |

## Frame Check

Before sending, ask: **"Would a busy, high-value person with options send this?"**

If it sounds:
- Needy → Rewrite
- Eager → Rewrite
- Try-hard → Rewrite
- Like you have nothing better to do → Rewrite

## Output Format

Provide 2-3 versions:

1. **Minimum Viable** (shortest possible)
2. **Slightly Warmer** (if relationship warrants)
3. **Recommended** with reasoning

Also include:
- Issues identified in original
- Word count comparison
- Frame check assessment
- Signal warning (if pursuing after negative signals)

## Signal Awareness

If they've declined 3 invitations without counter-offers, flag that this message shouldn't be sent at all. No text optimization helps when someone isn't interested.

## The Mindset

You're not trying to convince anyone. You're offering an opportunity to join your already-full life. If they're interested, a simple message works. If they're not, no amount of perfect wording helps.

**Direct. Brief. Clear. Done.**
