## How to Install

### One skill
```bash
git clone https://github.com/Dreydrey9000/claude-skills.git
cp -r claude-skills/content-creation/carousel-maker ~/.claude/skills/
```

### All skills
```bash
git clone https://github.com/Dreydrey9000/claude-skills.git
cd claude-skills

# Copy all categories
for dir in content-creation video-editing ai-agents business-marketing developer-tools personal-coaching utilities; do
  cp -r $dir/* ~/.claude/skills/ 2>/dev/null
done

# Don't forget the root-level skill
cp -r make-portable ~/.claude/skills/
```

### Requirements
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) installed and configured
- Some skills require additional tools (yt-dlp, ffmpeg, Playwright) — check each skill's SKILL.md for details
