# 20-Minute Course Builder

## What This Is

Ryan Magin's exact system for building a $29 digital course product in one afternoon. Not theory — this is the process that built Speed to Post, which sells 5-7 copies/day at $29 with a 20-30% conversion rate on a sales page that "looks like it was made in Microsoft Paint."

**Source:** Ryan Magin's workshops, team strategy meetings, and product recordings (Feb 2025).

## When To Use This Skill

- Drey says "build a course" / "make a product" / "I want to sell something" / "low ticket"
- A client needs to create their first digital product
- Someone has expertise but no product to sell
- Anyone asking how to monetize their audience with a simple offer

This skill has two distinct parts. Both are valuable. One is usable immediately, the other requires input.

## Real-Time Research (Perplexity Integration)

This skill uses the Perplexity MCP server to research real-time topics, trends, and content opportunities. This powers the FORMAT layer — especially reaction videos and POV content.

### When Research Triggers Automatically

**FORMAT 2 (Reaction Videos):** When generating reaction video ideas, the skill searches Perplexity for:
- Trending articles, tweets, and takes in the user's niche
- Controversial opinions or bad advice getting attention right now
- News or events their audience cares about today

Output includes: the specific article/tweet/post to screenshot, what to say about it, and the angle to take (agree loudly or disagree loudly).

**FORMAT 3 (POV / Screenshot Content):** When generating screenshot or "us vs them" ideas, search for:
- Popular posts with bad advice the user disagrees with
- Competitor content the user can counter
- Industry trends the audience has strong feelings about

**Daily Content Research:** When asked "what should I post today?" or "give me today's content":
1. Search Perplexity for what's trending in their niche RIGHT NOW
2. Cross-reference with their psychographic map (Layer 1)
3. Match to format types (Layer 2)
4. Write the scripts in their voice (Layer 3 + voice profile)
5. Tell them exactly what to screenshot, what to react to, what to say

### How To Trigger Research

- "What should I react to today?" → Perplexity search for trending takes in their niche
- "Find me something to screenshot" → Search for bad advice or hot takes to respond to
- "What's trending in [niche]?" → Real-time trend scan
- "Give me today's 3 posts" → Full research + scripting for all 3 daily formats
- Any content generation mode will auto-search when real-time relevance matters

## How To Use This Skill

### MODE 0: "Onboarding / Voice Capture" (Run First)

**Trigger:** "onboard [name]" / "capture voice" / "set up for [name]" / first time using skill for a new person

Walk through the intake process above. Steps:

1. Ask what materials are available (recordings, reels, content, docs)
2. Transcribe and process all provided recordings
3. Run the 20-question interview (live or from recording)
4. Pull and analyze their Instagram content
5. Review their existing sales pages and copy
6. Read their onboarding doc (if they have one)
7. Build the voice profile: `voice-profile-[name].md`
8. Test it: generate 3 sample CTA scripts and ask "does this sound like you?"
9. Refine until they say yes

**Output:** A voice profile file that all other modes reference. Stored in the skill directory or user's project folder.

### MODE 1: "Build My Course" (Default)

When triggered, walk through all 9 steps in order. Ask one question at a time to extract the user's one problem, expertise, and framework. Then generate:

1. The one-problem statement
2. The ChatGPT prompt to generate their outline
3. Loom recording instructions
4. The ChatGPT prompt for their sales page
5. Sales page humanization checklist
6. SamCart setup steps
7. ManyChat configuration
8. Order bump idea (next logical problem)
9. First CTA video script

### MODE 2: "Sales Page Generator"

When the user has a transcript, generate the sales page copy using Ryan's formula:
- Under 500 words
- Headline + subheadline + story + bullet points + price + CTA
- Written from their transcript so it sounds like them
- Include spots for iPhone Notes images

### MODE 3: "Order Bump Architect"

When the user has a product live, help identify and build order bumps:
- What's the NEXT problem after they use the main product?
- Same format: slides, Loom, 15-20 min
- Price at $19 (easy add-on)
- Stack up to 5 bumps (Maria Wendt method)

### MODE 4: "CTA Script Writer"

Generate daily CTA video scripts driving to the product. **Requires voice profile loaded.** Two formats:

**Format A: Full CTA Reel (60 sec)**
- Hook + value + CTA + trigger word
- Filmable in 10 minutes on iPhone
- Follow Ryan's CTA structure (see verbatim script above)
- Written in the person's actual voice (from voice profile — vocabulary, cadence, energy level)

**Format B: Big Brag Pitch Reel (15-30 sec)**
- Maria Wendt's formula: Big Brag + Only Next Step
- Lead with a specific result, end with one CTA
- No explanation. Just result → action.
- Must use THEIR specific results and numbers, not generic ones

### MODE 6: "Pinned VSL Builder"

Build the Jason Capital-style VSL to pin to Instagram profile:
- Structured pitch video (2-5 min) following JC's framework
- Hook → pace → credibility → problem → solution → offer → close
- Ends with ManyChat trigger word
- Pin to position 1 on grid — this is the 24/7 salesperson

### MODE 5: "Product Ladder Planner"

Map out the full product ladder:
- What problem does product 1 solve?
- What's the next logical problem?
- Build the sequence: each product feeds the next
- Price progression: $29 → $19 bump → $29 next product → higher tier

### MODE 7: "Today's Content" (Perplexity-Powered Daily Engine)

**Trigger:** "what should I post today?" / "give me today's content" / "what should [name] react to?" / "find me something to screenshot"

This mode uses Perplexity to research real-time topics and generates all 3 daily posts with specific, actionable direction:

1. **Research phase** — Search Perplexity for trending topics, articles, tweets, and takes in the user's niche. Cross-reference with their psychographic map.
2. **Post 1 (YAP Reel)** — Pick the most relevant trending topic → write a script in their voice (from voice profile) with their angle on it
3. **Post 2 (Reaction Video)** — Find a specific article, tweet, or bad take to react to → provide the exact link/screenshot + what to say + whether to agree or disagree and why
4. **Post 3 (POV/BTS/Screenshot)** — Find a rallying opportunity → tell them exactly what to screenshot or film for the "us vs them" moment

**Output for each post includes:**
- The hook (first 3 seconds)
- What to say (in their voice, not template language)
- What to screenshot or show on screen (with link if applicable)
- CTA or no CTA (and which trigger word if yes)
- Which funnel tier it serves (top/middle/bottom)

**Can also be used for:**
- "Find me 5 things to react to this week" → batch research
- "What's my audience talking about right now?" → trend scan
- "Give me screenshot content" → find specific posts/comments to respond to
