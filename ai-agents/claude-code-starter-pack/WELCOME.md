Most people use Claude Code at 10% capacity. They type prompts, click approve, and wonder why it feels like a chatbot. It's not a chatbot. It's an **Agent Operating System** — and this pack turns it on.

## What Just Happened

You now have:

| Component | What It Does |
|-----------|-------------|
| **Directory Structure** | `~/.claude/` organized: hooks, skills, references, scripts, agents |
| **Hooks** | Lifecycle events that make Claude Code deterministic, not random |
| **CLAUDE.md Template** | Your personality file — Claude thinks like YOU, not a generic bot |
| **Context Diet System** | Sessions use 8% context on setup instead of 27%. No more mid-build crashes. |

1. **Customize `~/.claude/CLAUDE.md`** — Tell Claude who you are, how you work, what you're building

3. **Add MCP servers** — Gmail, Notion, research tools. Each one gives Claude a new superpower.
4. **Start building** — Type `claude` and go
