# Viral Accelerator Skill

## Purpose
Generate 90 film-ready viral video topics for any client using ATTN University frameworks (HEM, CUB, HSP, LSL, AEP) combined with the 3S Method (Symptoms, Solutions, Suspicions). Transform raw client intelligence into a complete filming blueprint that produces 20M+ views.

## When to Use This Skill
- When user wants to create viral content for a client
- When you have client materials (sales calls, transcripts, podcasts, website, recordings)
- When you need systematic viral topic generation
- When building content strategy for client acquisition

## What This Skill Does
1. Extracts deep client intelligence using 3S Method
2. Maps to Ryan Magin's viral frameworks
3. Generates 90 topics by filming setup
4. Creates exact directing questions
5. Applies HEM distribution (80/15/5)
6. Runs CUB quality filter
7. Structures with HSP
8. Incorporates LSL proof elements
9. Uses AEP targeting strategy
10. Provides complete filming blueprint + release plan

## EXAMPLES

### ❌ BAD TOPIC:

**Hook:** "Leadership is important for business growth"

**Problems:**
- Fails CUB: Vague, no proof, generic
- No HSP: Where's the story? What's the prescription?
- No LSL: Nothing to show
- Wrong AEP: Talking about business, not client interests
- Not Observable: "Leadership" is abstract
