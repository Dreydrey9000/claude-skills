# Content Repurposer

Transform a single piece of content into multiple platform-native formats that stand alone.

## When to Use

- Converting a podcast/video transcript into social content
- Adapting a blog post for multiple platforms
- Creating a content series from one source
- Maximizing ROI from existing content

## Platform-Native Adaptation

Each platform is a different medium. Don't just reformat—reimagine.

### Twitter/X Threads
- Hook tweet must stand alone
- Each tweet adds value independently
- Thread builds to conclusion
- End with CTA or engagement prompt

### Instagram Carousels
- First slide = scroll-stopping hook
- 6-10 slides of value
- Last slide = clear CTA
- Caption adds context, not repetition

### LinkedIn Posts
- Story format works best
- Professional but personal tone
- Line breaks for readability
- 1200-1500 characters optimal

### TikTok/Reels Scripts
- First 1 second is everything
- Pattern interrupt immediately
- Fast pacing, no fluff
- Clear single message

## The Repurposing Process

1. **Identify Core Ideas** - Extract 3-5 strongest concepts from source
2. **Match to Platform** - Which ideas fit which platform best?
3. **Adapt Format** - Restructure for platform conventions
4. **Maintain Voice** - Keep brand consistency
5. **Ensure Standalone** - Each piece works without source context

## Output Format

For each source piece, provide:

**Twitter Thread:**
- Hook tweet
- 5-8 thread tweets
- CTA tweet
- Standalone variation (single viral tweet)

**Instagram Carousel:**
- Hook slide text
- Content slides (text for each)
- CTA slide text
- Caption (with hashtag suggestions)

**LinkedIn Post:**
- Full post with line breaks
- Hook, story, lesson structure

**Short-Form Script:**
- Hook (first 3 seconds)
- Body (15-45 seconds)
- CTA (final 5 seconds)
- Duration estimate

**Repurposing Notes:**
- What worked best from source
- Future content angles to explore

## Anti-Patterns

- Copy-pasting without adaptation
- Content that "feels repurposed"
- Ignoring platform conventions
- Losing core message in translation
- Same hook across all platforms

## Key Principle

**Repurposed content should feel native.** If someone on Twitter reads your thread, they shouldn't think "this was clearly from a podcast." Each piece earns its place on that platform.
